## Mac OS

### Créer l'environnement
```bash
python3 -m venv venv
```
### Activer l'environnement
```bash
. venv/bin/activate
```

### Installer les packages
```bash
pip install -r requirements.txt
```

### Lancer le serveur
```bash
python3 app.py
```

### Tests
```bash
python3 -m unittest Test.Test_Movie
```
